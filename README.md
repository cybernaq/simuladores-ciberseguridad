# 🛡️ Simuladores Abiertos de Ciberseguridad

¡Bienvenido al repositorio oficial del proyecto de Simuladores de Ciberseguridad de **Cybernaq**!

Este repositorio contiene el código fuente (HTML, CSS y JavaScript) de herramientas interactivas diseñadas para la enseñanza práctica de conceptos de ciberseguridad, recolección de información (OSINT) y escaneo de redes.

## 🚀 Ver y Usar los Simuladores

Si eres estudiante o simplemente quieres **utilizar** las herramientas, no necesitas descargar este código. Puedes acceder directamente a la plataforma en vivo aquí:

👉 **[https://cybernaq.github.io/simuladores-ciberseguridad/](https://cybernaq.github.io/simuladores-ciberseguridad/)**

## 🤝 Guía para interesados en Contribuir

Este es un proyecto de código abierto y la comunidad es bienvenida a proponer mejoras, optimizar la lógica de las simulaciones o crear nuevos módulos. 

Para garantizar la seguridad, la estabilidad del material educativo y evitar código malicioso, **ningún cambio se aplica automáticamente**. Todas las aportaciones deben pasar por un proceso de revisión y aprobación del administrador.

### Pasos para enviar tu aporte:

1. **Haz un Fork del proyecto:** Crea una copia de este repositorio en tu propia cuenta de GitHub haciendo clic en el botón "Fork" (arriba a la derecha).
2. **Crea una nueva rama (Branch):** En tu repositorio copiado, crea una rama específica para tu mejora. Ejemplo: `git checkout -b mejora-analisis-nmap`.
3. **Realiza tus Commits:** Haz los cambios en el código y guárdalos (commits) en tu copia. Asegúrate de que el código esté limpio y comentado si es necesario.
4. **Abre un Pull Request (PR):** Vuelve a este repositorio original (Cybernaq) y abre una solicitud de "Pull Request" comparando tu rama con nuestra rama `main`.
5. **Proceso de Revisión:** 
   * Describe detalladamente en tu Pull Request qué cambiaste y por qué.
   * El administrador revisará línea por línea tu código.
   * Si todo es correcto y seguro, **aprobaremos y fusionaremos** tu aporte. Si hay detalles a corregir, te dejaremos comentarios para que los ajustes.
